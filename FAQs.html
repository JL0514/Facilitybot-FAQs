Agila Subic FAQ

Frequently Asked Questions

For Subtenants

Go to: Agila Subic Bot → Menu → Service Request → Create Service Request → Select "Service Request."

This is for:

  • Repair & maintenance requests (e.g., electrical, plumbing, aircon, lighting)
  • Facility-related issues (e.g., cleanliness, pest control, space concerns)
  • General assistance requests

Timeline:

  • File requests at least 1 day in advance to allow for processing and assignment
  • For urgent matters, please call the Agila Facility Management Office after filing in the bot for faster action

a. Visitor Entry Request / Delivery

Navigate to: Agila Subic Bot → Menu → Service Request → Create Service Request → Select "Visitor Entry Request / Delivery."

Purpose:

  • Guests or visitors (e.g., clients, auditors, business partners)
  • Deliveries (documents, office supplies, packages, equipment)
  • Vendors, suppliers, or contractors for meetings

Important:

  • All visitor details must match actual arrival (name, ID, plate number)
  • Mismatched details = delay or possible denial at gate
  • Frequent visitors (more than once per week) must attend Safety & Security Induction

Validity & Processing:

  • Valid for 3-7 days depending on request type
  • File at least 1 day before intended visit
  • Processing of requests is limited during weekends
  • Always present the Case ID at the gate for quick verification

b. Gatepass for Pull-Out

Navigate to: Agila Subic Bot → Menu → Service Request → Create Service Request → Select "Gatepass for Pull-Out."

Purpose:

  • Removal of company-owned or personal items, equipment, tools, and materials
  • Covers both temporary pull-out (for repair/servicing) and permanent removal

Schedule:

  • Monday-Friday: 8:00 AM - 5:00 PM
  • Saturday: 8:00 AM - 12:00 NN only
  • ❌ No Pull-Outs allowed on Sundays

Requirements:

  • Proper documentation and approvals from concerned departments
  • Clear description of items (serial numbers, quantity, purpose)
  • Security has the right to inspect and validate items against approved gatepass

c. HSE Induction

Mandatory for:

  • Contractors working onsite for projects or repairs
  • Vendors or service providers with regular presence inside the shipyard
  • Visitors engaged in activities that expose them to operational areas

Covers:

  • Safety protocols, security policies, and emergency response procedures
  • PPE usage, restricted areas, and conduct rules

How to Request: File through Agila Subic Bot → Service Request → HSE Induction

  • Always carry a valid Company ID or government-issued ID
  • Comply with dress code and wear PPE in operational zones
  • Vehicles must use the registered plate number in the request
  • Any discrepancies (plate mismatch, undeclared visitor, unapproved pull-out) = entry denial or delay
  • Follow Agila Subic's security, HSE, and facility rules at all times

For Contractors / Vendors / Suppliers

a. Visitor Entry Request / Delivery

Go to: Agila Subic Bot → Menu → Service Request → Create Service Request → Select "Visitor Entry Request / Delivery."

Applies to:

  • Contractors mobilizing manpower for work
  • Vendors delivering supplies, tools, or spare parts
  • Suppliers entering for meetings or deliveries

Details must be accurate (names, ID, vehicle plates). Incorrect details will result in delayed or denied entry.

Validity & Processing:

  • Pass valid for 3-7 days
  • File at least 1 day in advance
  • Frequent entries = mandatory Safety & Security Induction

b. Gatepass for Pull-Out

Go to: Agila Subic Bot → Menu → Service Request → Create Service Request → Select "Gatepass for Pull-Out."

For removing:

  • Tools, equipment, or spare parts after project completion
  • Damaged/defective items for offsite repair

Schedule:

  • Monday-Friday: 8:00 AM - 5:00 PM
  • Saturday: 8:00 AM - 12:00 NN
  • ❌ No Sunday Pull-Outs

Requirements:

  • Approved gatepass form with complete item details
  • Proper supporting documents (delivery receipt, work permit, or project approval)
  • Items subject to inspection by Security before exit

c. HSE Induction (Mandatory)

All contractors, subcontractors, and manpower personnel must undergo HSE Induction before starting work.

Covers:

  • General site safety, PPE requirements, and emergency protocols
  • Security access policies, prohibited acts, and reporting guidelines

No induction = no work access allowed.

Before Work:

  • Secure all permits (Access, Gatepass, Work Permits)
  • Ensure all staff are inducted and registered
  • Provide complete PPE for personnel

During Work:

  • Always wear PPE
  • Follow Agila Subic's HSE and security rules
  • Cooperate with inspections and monitoring

After Work:

  • Clear all work areas
  • File Gatepass for removal of tools and equipment
  • Return visitor passes and IDs before leaving premises
  • File requests ahead of time (at least 1 day)
  • Weekend processing is limited, so plan deliveries/pull-outs earlier
  • Keep copies of Case ID for reference at the gate
  • Non-compliance = possible delays, penalties, or blacklisting from future projects
